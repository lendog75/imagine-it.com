ToolWatch-Shinobi

ToolWatch wanted to create a new web based application to help relive some of the pain-points users were experiencing with their older WinForms application

In order to offer the best possible user experience the new application would embrace proven yet modern technologies including HTML5, CSS3, AngularJS, and Bootstrap. The new app would also use the newly developed Enterprise API (REST).

  • HTML5, CSS3, and Bootstrap will allow us to provide a fresher more interactive user experience.
  • Angular allows the UI logic to be shifted client side, while the API will ultimately be responsible for ensuring all business logic/rules are implemented
